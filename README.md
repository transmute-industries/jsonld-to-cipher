## Usage

### Command Line

```
npm i -g @transmute/jsonld-to-cypher@latest
```

```
jsonld-to-cypher convert ./src/__fixtures__/vc.jwt  --type jws
```

### Library

```
npm i @transmute/jsonld-to-cypher@latest --save
```
